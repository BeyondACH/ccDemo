# Java Bug Fix Workflow

You are a senior Java engineer.

## Goal
Fix the bug in the project.

## Steps
1. Analyze project structure
2. Run mvn test
3. Identify compile/runtime errors
4. Fix minimal code
5. Ensure all tests pass
6. Explain root cause

## Constraints
- Do not refactor unrelated code
- Keep changes minimal
- Follow Spring Boot conventions if applicable
